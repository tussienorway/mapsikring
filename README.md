# Mapsikring

Profesjonell tjeneste for sikring, flytting og optimalisering av Google Maps-profiler for norske bedrifter.

## Nettside

- **Live demo**: [https://tussienorway.github.io/mapsikring/](https://tussienorway.github.io/mapsikring/)
- **Repo**: [github.com/tussienorway/mapsikring](https://github.com/tussienorway/mapsikring)

## Prosjektmål

- Hjelpe norske virksomheter med trygg flytting av Google Business Profile
- Optimalisere lokal synlighet på Google Maps
- Beskytte profiler mot uautorisert tilgang
- Generere leads gjennom profesjonell landingsside
